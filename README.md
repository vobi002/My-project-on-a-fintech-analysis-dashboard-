# FinTech Global Transaction Analysis Dashboard
## Project Overview
This project involved analyzing a comprehensive digital payment dataset to uncover user behavior patterns, channel preferences, and system performance metrics. Using **Power BI** and **Excel**, I transformed raw transaction data into an interactive executive dashboard.

### 🚀 Key Business Insights
* **[span_0](start_span)Transaction Volume:** Successfully tracked and visualized over **$2.02M** in total transaction volume across global markets[span_0](end_span).
* **[span_1](start_span)User Engagement:** Analyzed activity for **5,232 total users**, identifying **London and Seattle** as top-performing cities[span_1](end_span).
* **[span_2](start_span)Channel Performance:** Identified a preference for mobile transactions, which accounted for **3,040** completed actions compared to **2,928** on desktop[span_2](end_span).
* **[span_3](start_span)Revenue Drivers:** Determined that **Top-Ups and Transfers** are the primary revenue contributors, generating the bulk of the **$13.10K** in total fees[span_3](end_span).
* **[span_4](start_span)System Health:** Pinpointed **288 failed** and **272 pending** transactions, providing a roadmap for technical optimization[span_4](end_span).
